# HCMC_Real_Estate_Analytics
An automated data pipeline and machine learning regression model designed to solve market asymmetry and optimize pricing strategies.
* **Role:** Data & Product Analyst
* **Core Tech:** Python (Pandas, NumPy, Scikit-learn), Groq AI API, Feature Engineering, Random Forest, XGBoost.
* **Key Achievements:**
  * Engineered automated web scraping scripts to crawl property listings from major Vietnamese real estate platforms including **Batdongsan**, **Homedy**, and **Nhadatcanban**.
  * Leveraged **Groq AI API** within the pipeline to clean, parse, and impute missing variables from the raw crawled datasets.
  * Conducted exploratory data analysis and segmentation to identify high-potential strategic market gaps for digital product insights.
  * Co-developed machine learning regression models (**Random Forest, XGBoost**) focused on feature engineering to enhance valuation accuracy.
